## JSON DBs for Projects

1.  **printers.json** - List of printers/MFP with cartridges list. \
    Format:
    ```json
    {
      "manufacturer": {
        "model": ["cartridges", "list"]
      }
    }
    ```
2. **ip-phones.json** - List of IP-Phones. \
   Format:
   ```json
   {
     "manufacturer": ["models", "list"]
   }
   ```
